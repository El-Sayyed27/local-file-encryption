# 🔐 Secure File Encryptor

A lightweight Python-based file encryption and decryption tool using **Fernet cryptography** with a simple **Tkinter GUI**.  
It allows users to securely encrypt any file type and decrypt it back to its original form with ease.

---

## ✨ Features

- 🔒 Encrypt any file type
- 🔓 Decrypt encrypted files (.enc)
- 🧠 Automatic key generation
- 🗂 Secure key storage (hidden file support)
- 📁 GUI file selection using Tkinter
- ⚡ Simple and beginner-friendly design
- 📦 Lightweight and fast execution

---

## 🛠 Requirements

Install the required dependency:

- `pip install cryptography`
